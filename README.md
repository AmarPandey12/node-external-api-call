# node-external-api-call
Making third party api calls in node js

# This poc project helps understanding the concept behind API calls by calling 3rd party service
This small project helps you understand the basic setup for node project.
Here I have used only one npm package which is Axios to make 3rd party api call.
Provising api endpoints for your service and calling other services is core for software application and this cooked recipe you will guide to setup the application and make GET call.
Response is handled alongwith error in case.
